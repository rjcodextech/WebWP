### Description
WebWP is a versatile WordPress starter theme tailored for developers and designers seeking to create customizable layouts and structures. This theme comes with built-in support for footer and sidebar widgets, making it easy to extend functionality and enhance the user experience. It is perfect for building unique websites with flexible design options.

## Installation
1. Download the WebWP theme zip file.
2. In your WordPress admin dashboard, go to **Appearance > Themes**.
3. Click **Add New** and then **Upload Theme**.
4. Choose the downloaded zip file and click **Install Now**.
5. Activate the theme once installed.

---

## Customization
- Navigate to **Appearance > Customize** in the WordPress admin to start customizing your theme settings, including site identity, colors, menus, and more.
- Use the **Widgets** section under **Appearance** to add custom widgets to the footer and sidebar.
